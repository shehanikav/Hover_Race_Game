# 3D Hover Car Racing Game 🚗💨

A 3D Hover Car Racing Game developed using C++ and TL-Engine for the Gaming Concepts module at the University of Lancashire (via UCL).

## Features

* WASD player movement
* Waypoint-based NPC AI
* Collision detection and bounce physics
* Boost mechanics
* Lap and checkpoint system
* Multiple camera views
* Particle systems and visual effects
* Frame-time based movement
* File loading system
* Game states using enums

## What I Learned

This project helped me improve my understanding of:

* Real-time game programming
* Physics and momentum systems
* Vector-based movement
* Object-oriented programming
* Game architecture and state management
* Debugging and gameplay balancing
